# Data

This project uses the **Seattle Food Establishment Inspection Data**, a
public dataset available from Seattle's open data portal:

https://data.seattle.gov/

Search for "Food Establishment Inspection Data," download the CSV, and
place it here as:

```
data/Food_Establishment_Inspection_Data.csv
```

The raw file isn't committed to this repo since it's large and the source
updates regularly — download a fresh copy to reproduce the analysis exactly
as run, or use whatever snapshot is currently available for a similar
(though not identical) result.
